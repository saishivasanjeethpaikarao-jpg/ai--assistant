# Jarvis/Siri Improvement Plan

This project is evolving into a Jarvis/Siri-style personal assistant in clear phases.

## Phase 1 — Voice persona and app readiness
- Add a consistent assistant persona and name (`Jarvis`).
- Use wake-word voice flow and voice chat in the app.
- Keep replies concise and helpful.
- Improve app startup greeting and UI status.

## Phase 2 — Personal memory and profile
- Save and recall user facts in `memory/memory.json`.
- Store active profile details through Firebase login.
- Include user memory context in AI prompts.
- Add commands like `remember`, `recall`, and `forget`.

## Phase 3 — Smart skills and automation
- Add better app/file opening and search actions.
- Support safe PowerShell execution and system controls.
- Add reminders, notes, and proactive suggestions.
- Add daily briefing and reminder support.
- Add richer task flows for follow-up questions.

## Phase 4 — App polish and packaging
- Improve the Kivy desktop UI and voice activity feedback.
- Build a Windows executable and installer.
- Add branding, icons, and tray/background support.
- Add richer assistant persona prompts and an onboarding flow.
