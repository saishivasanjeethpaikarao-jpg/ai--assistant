# 🎯 Quick Command Reference - NSE/BSE Trading

## Stock Price Lookup
```
"stock price RELIANCE"
"get price TCS"
"price INFY"
```

## Search Stocks
```
"search stock reliance"
"find TCS"
"find indian oil"
```

## Compare Exchanges
```
"compare RELIANCE exchange"
"compare NSE BSE INFY"
```

## Market Analysis
```
"market gainers"           → Top 10 gaining stocks
"market losers"            → Top 10 losing stocks  
"market summary"           → Overall market status
```

## Exchange Suffixes

**NSE (National Stock Exchange) - DEFAULT**
- Symbol: `RELIANCE` or `RELIANCE.NS`
- Example: `"stock price RELIANCE"`

**BSE (Bombay Stock Exchange)**
- Symbol: `RELIANCE.BO`
- Example: `"stock price RELIANCE.BO"`

## Tips

1. **Use Stock Symbols** - Faster: `"price RELIANCE"` vs `"price reliance industries"`
2. **NSE is Default** - `RELIANCE` = `RELIANCE.NS` automatically
3. **For BSE** - Add `.BO`: `"price RELIANCE.BO"`
4. **Price Alerts** - `"add INFY at 1500"` alerts when price hits ₹1,500
5. **Portfolio Math** - Auto-calculates gains/losses from cost basis
6. **Market Hours** - Best data during 9:15 AM - 3:30 PM IST
7. **All Data Persists** - Watchlist & portfolio saved automatically
