PART 2: API ENDPOINTS — add to backend/api_server.py
═══════════════════════════════════════════════════════════

POST /vibe/code
  Request:  { request: str, voice: bool, history: list }
  Response: { 
    agent_name: str, agent_emoji: str,
    code: str, language: str,
    explanation: str, canRun: bool,
    dependencies: list[str],
    execution_id: str,
    confidence: float
  }

POST /vibe/run
  Request:  { code: str, language: str }
  Response: { 
    output: str, error: str, 
    success: bool, execution_time: float 
  }

POST /vibe/fix
  Request:  { code: str, error: str }
  Response: { fixed_code: str, explanation: str }

GET /vibe/agents
  Response: [{ name, emoji, description, keywords, example }]

POST /vibe/chat
  Request:  { message: str, code_context: str }
  Response: { reply: str }
  # For asking questions about generated code
