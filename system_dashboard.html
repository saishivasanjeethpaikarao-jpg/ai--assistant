Request Processing Workflow

How It Works

Stage Layer Action Output
1 Intent Detector Classifies input as COMMAND, GOAL, or CHAT Type + Complexity
2-3 Planner & Critic Creates 3-5 step plan and validates Optimized Plan
4-6 Executor & Safety Converts steps to commands and validates safety Safe Commands
7-8 Reflection & Memory Evaluates outcome and stores learning Status + Learning
9 Re-Planner If failed, creates improved plan Recovery Plan
12 Orchestrator Coordinates all layers and returns result Final Output
